\# Data Dictionary



This document explains the fields used in the loan risk prediction dataset.



| Field Name | Definition | Data Type | Example | Notes |

|---|---|---|---|---|

| Age | Age of the loan applicant | Integer | 35 | Used for applicant profile analysis |

| Income | Annual income of the applicant | Numeric | 75000 | Used to evaluate repayment capacity |

| LoanAmount | Requested loan amount | Numeric | 25000 | Used for exposure and portfolio analysis |

| CreditScore | Credit score of the applicant | Integer | 720 | Used for risk segmentation |

| YearsExperience | Total years of work experience | Integer | 8 | Used to evaluate applicant stability |

| Gender | Gender of the applicant | Text | Male / Female | Used for demographic analysis |

| Education | Education level of the applicant | Text | Graduate | Used for segmentation |

| City | Applicant city | Text | Tampa | Used for geographic analysis |

| EmploymentType | Employment category of applicant | Text | Salaried / Self-employed | Used for approval pattern analysis |

| LoanApproved | Loan approval result | Integer / Boolean | 1 or 0 | 1 = Approved, 0 = Not Approved |
